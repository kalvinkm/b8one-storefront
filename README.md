# 🛍️ B8One Storefront

Aplicação desenvolvida como desafio técnico para vaga de Front-end Júnior.

---

## 🚀 Tecnologias

- React
- TypeScript
- Vite
- CSS
- ESLint + Prettier

---

## 📐 Arquitetura

O projeto segue princípios de Clean Architecture, com separação em camadas:

- domain
- application
- infra
- presentation

---

## ▶️ Como rodar

```bash
npm install
npm run dev
```

---

## 📌 Funcionalidades

- ✅ Página de ofertas
- ✅ Banner responsivo com CTA
- ✅ Scroll suave para seção de produtos
- ✅ Seção de produtos estruturada
- ✅ Grid responsivo de produtos
- ✅ Mock de produtos para validação de layout

### 🚧 Em desenvolvimento
